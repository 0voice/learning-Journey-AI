# 模型优化与加速工具
## TensorFlow Lite：

- 用途：用于在移动设备或边缘设备上运行深度学习模型的轻量级版本。

- 安装：

```bash
pip install tflite
```

## NVIDIA Apex：

- 用途：NVIDIA 提供的库，优化 PyTorch 中的混合精度训练。

- 安装：

```bash
pip install apex
```

## Horovod：

- 用途：分布式深度学习库，用于加速 TensorFlow、Keras 和 PyTorch 的训练。

- 安装：

```bash
pip install horovod
```
